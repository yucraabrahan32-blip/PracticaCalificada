¿Qué es una estructura de datos?
Una estructura es la manera de organizar y poder almacenar los datos en la computadora para que as pueda se mas eficiente y rápido 

¿Cómo almacenan datos las computadoras?
La computadora traduce todo el lenguaje de 0 y 1 

Tipos de estructuras de datos
Existen dos tipos que son los estáticos y los dinámicos 

Estructuras de datos estáticas:
En las estructuras de datos estáticos no se pueden cambiar cuando el programa esta en ejecución 

Estructuras de datos dinámicos:
Puede permitir cambiar datos cuando el programa esta en ejecución 

Estructuras de datos dinámicas lineales:
Listas enlazadas : son una colección de nodos donde cada nodo tiene un dato 
Pilas : es donde el primero en entrar y en el ultimo en salir 
Colas : es donde el primero en entrar es el primero en salir 

Estructuras de datos dinámicas no lineales: 
Arboles : es una estructura jerárquica en la cada nodo tiene un valor y  una lista  o un tipo especial de árbol binario  en q cada nodo tiene máxima capacidad de 2 hijos ..

Estructuras de datos dinámicas no lineales:
Grafos: una colección de nodos y arcos que se conectan pares de nodos 
Tablas hash : una estructura  que asocia claves a valores
